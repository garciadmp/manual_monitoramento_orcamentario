# Manual de Monitoramento Orçamentário

## Apresentação

Este repositório contém os arquivos, códigos e conteúdos que dão suporte ao **Manual de Monitoramento Orçamentário**, disponível em:

https://garciadmp.github.io/manual_monitoramento_orcamentario/

O manual foi desenvolvido com o objetivo de oferecer um guia prático para que os órgãos da Administração Direta e Indireta da Prefeitura de São Paulo, com compromissos pactuados no Programa de Metas 2025-2028, possam prestar informações sobre a execução orçamentária relacionada ao referido instrumento de planejamento.

## Objetivos-
O projeto busca:

* Sistematizar procedimentos para o monitoramento da execução orçamentária
* Facilitar o acesso e a interpretação de dados públicos
* Apoiar análises aplicadas à gestão pública e ao controle social
* Contribuir para o desenvolvimento de capacidades analíticas no setor público

## Estrutura do repositório

O repositório está organizado da seguinte forma:

* `docs/`
  Contém os arquivos renderizados que compõem o site publicado via GitHub Pages.

* `scripts/` ou `R/`
  Scripts utilizados para coleta, tratamento e análise dos dados orçamentários.

* `data/`
  Bases de dados utilizadas ou geradas ao longo do projeto (quando aplicável).

* `images/` ou `figures/`
  Elementos gráficos utilizados no manual.

* Arquivos de configuração
  Incluem `.Rprofile`, `.gitignore` e outros arquivos necessários para reprodutibilidade do ambiente.

## Como utilizar

O conteúdo pode ser utilizado de duas formas:

1. Navegação direta pelo site publicado
   Ideal para leitura contínua e consulta metodológica.

2. Reprodução local
   Para usuários interessados em adaptar ou expandir as análises:

   * Clone o repositório
   * Configure o ambiente R conforme os requisitos
   * Execute os scripts disponíveis

## Requisitos

Para reprodução dos códigos, recomenda-se:

* R (versão atualizada)
* RStudio (opcional, mas recomendado)
* Pacotes usuais de ciência de dados (por exemplo: tidyverse, httr, jsonlite, sf, entre outros conforme especificado nos scripts)
